# Antigravity에 Unity MCP 붙이기

## uv 설치

MCP는 다들 파이썬으로 동작하게 만들고, 파이썬의 uv 패키지가 필요하다고 한다.

우선 uv 패키지를 설치하자

```bash
pip install uv
```

Unity의 Package Manager에서 url로 import
```
https://github.com/CoplayDev/unity-mcp.git?path=/MCPForUnity#main
```

Window -> MCP for Unity -> Toggle MCP Window -> Server Start, Client Configure 설정

Antigravity에선 우상단 MCP Servers에서 Manager MCP Servers를 누르면 Unity의 MCP Configure 창을 볼 수 있음
