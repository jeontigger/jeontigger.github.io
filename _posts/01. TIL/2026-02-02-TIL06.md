## 2. MCP란?

AI 모델들과 어플리케이션을 연결하는 프로토콜

Claud, Gemini, GPT등 모델이 많은데, 각각의 모델이 특정 어플리케이션과 소통하는 방식이 다름. 이를 통일하기 위해 등장한 것이 MCP임.

MCP Host: AI 모델

MCP Client: 어플리케이션

MCP Server: 어플리케이션과 AI 모델을 연결하는 서버

를 뜻함.

MCP 서버는 AI에게 3가지를 제공함

1. 리소스: 어플리케이션이 갖고있는 데이터

2. 도구: AI가 실행할 수 있는 함수나 기능 (예: API 호출, DB 조회 등)

3. 프롬프트: AI에게 미리 정의된 지침이나 템플릿을 제공함

이로써 AI가 더 컨텍스트를 쉽고 빠르게 파악하고, 더 정확한 결과를 제공할 수 있게 됨

## 3. Antigravity

### Skills

Skills는 skill에 해당하는 프롬프트를 입력하여 agent가 해당 분야의 전문가가 되도록 만드는 것.

Cluad 뿐만 아니라, Antigravity에게도 Skills가 있음.

https://github.com/sickn33/antigravity-awesome-skills

```bash
# Easiest: npx installer (clones to ~/.agent/skills by default)
npx antigravity-awesome-skills

# Or clone manually
git clone https://github.com/sickn33/antigravity-awesome-skills.git .agent/skills
```
으로 받을 수 있음

받고나면 .agent/skills 폴더에 들어가게 됨.

Antigravity 기준으로 자연어로 어떤 스킬 써줘 하면 된다고 함
